# HypernetsFR

This repository contains the code used to analyse the output from the HYPERNETS stations in France. It also provides figures from the testing of this code, as well as those used in the publication for the related validation study.

## Structure

All code may be found in the `/code/` folder. The `/output/` of this code, and the `/figures/` it generates are likewise to be found in reasonably named locations. Any required `/meta/`data are stored here as well.

## Data

Note that the base data files in .csv and .sql are generated by the `__Hypernets_matchups__` software, which is external to this project.
